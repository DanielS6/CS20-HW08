Describe the API you selected and what it does

The API I selected in the MediaWiki Action API, which allows clients to interact with a MediaWiki wiki programmatically, rather than requiring loading the user interface. It is used for everything from making edits, to changing account permissions, to even deleting or undeleting a page. See the full documentation on MediaWiki.org for further details.

Cite the website where you found it

The website where I found the API endpoint that I was accessing is on the English Wikipedia, at https://en.wikipedia.org/w/api.php, which also provides human-readable documentation for the different ways that the API can be used.

Describe the options you used for the API request

The options that I used are as follows:

Name Value Description
action query Use the `query` module, which is the primary module for retrieving data.
meta siteinfo Tell the `query` class to retrieve the `siteinfo` meta information, which is used for information about the target site.
siprop statistics Tell the `siteinfo` class to get information about the 'statistics' property (the `si` at the start of the option name is to disambiguate the option from a 'prop' option in any other class).
format json Return the data as a JSON object, rather than one of the other supported formats.
formatversion 2 Use the second version of the JSON format that the API provides, which notably uses true/false values for boolean properties rather than a key with an empty string value for true and being absent for false.
origin * Tell MediaWiki that the request is coming from an unauthenticated source and should not be checked against CORS (which also prevents using some parts of the API which require authentication).
Give two applications where this API would be helpful

This API would be useful for applications that want to monitor recent changes for vandalism, or for those who want to retrieve the text of articles to be used somewhere else.
